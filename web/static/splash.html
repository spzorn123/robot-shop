Welcome to Stan's Robot Shop (monitored by New Relic)

This is a simple example microservices ecommerce application. It has been built using various technologies:

  • AngularJS (1.x)
  • Nginx
  • NodeJS
  • Java
  • Python
  • Golang
  • PHP (Apache)
  • MongoDB
  • Redis
  • MySQL
When deployed with New Relic agents, these technology stacks will be automatically detected and monitored, all with minimum configuration. Every request will be traced end to end.

To find out more visit the New Relic site.

All the code is available on Github.
